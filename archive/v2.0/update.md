# SeaScribe 更新日志

## v2.0

### 布局双模式
- 新增列表/网格双模式切换（`☰` / `⊞` 按钮）
- 网格模式：卡片方阵排列，适合投屏浏览
- 列表模式：卡片横向排列，答案跟随题目下方，CSS Multi-Column 原生列优先填充
- 切换模式时伴有过渡动画
- 网格和列表各自独立记忆列数和字号

### 配置系统重构
- 所有默认参数从配置文件读取，零硬编码兜底
- 配置文件统一在 `config/` 目录，按学科分文件夹
- 网格模式、列表模式的列数和字号分别可配置
- 配置文件添加详细中文注释和取值说明

### 英语插件增强
- 默认列表模式，2 列显示
- 按钮「🔍 从服务器获取」进入时自动扫描 `data/english/` 目录
- 支持本地 `.xlsx` / `.csv` 文件导入
- 听写列和答案列可通过步进器调整，切换即时生效

### 化学插件
- 默认网格模式，4 列显示
- 价层电子式和轨道式同时展示
- H~Kr 36 元素，范围选择器

### 界面优化
- 答案文字正常换行（`pre-wrap`），去掉虚线分隔
- 卡片序号跟随字号缩放
- 控制栏紧凑化，增大卡片空间
- 随机点名动画约 2 秒
- 默认浅色主题
- favicon 图标

### 项目结构
- 目录重构：`main/`（代码）、`config/`（配置）、`plugins/`（插件）、`data/`（数据）
- `server.py` 提供通用 `/api/<name>-files` 接口，禁止目录浏览
- URL hash 路由：`#/english`、`#/chemistry`
- 更新日志独立为 `update.md`，点击「更新日志」按钮动态加载

### 开屏动画
- 启动时显示品牌动画
- 逐项检查：配置加载、插件注册、页面元素
- 点击任意位置关闭
