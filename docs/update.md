# SeaScribe 更新日志

## v3.3.0

### 化学数据文件化
- 化学元素数据迁移至独立 CSV 文件（`data/chemistry/elements.csv`），可用 Excel 或记事本编辑
- 化学页新增文件选择功能：🔍 扫描服务器目录、下拉框切换文件、📥 从本地上传 CSV
- 打开化学页自动加载默认数据，无需手动操作

### 关于页面
- 顶栏新增 ℹ️ 关于按钮，点击弹窗展示项目说明（内容来自 README）

### 界面调整
- 答案卡片标签与弹窗标题由红色改为正文色，视觉更统一
- Logo 切换为 PNG 格式

### 点名动画
- 点名定格时触发彩色纸屑特效（canvas-confetti）
- 优化动画时序，整体节奏更紧凑

## v3.2.0

### 名单系统重构
- 名单从 JS 硬编码改为 CSV 文件，放在 `data/stdlist/` 下即可自动识别
- 顶栏下拉框自动扫描 `/api/stdlist-files`，无需修改代码
- CSV 第 2 列支持头衔，点名定格后在全屏显示
- 点名动画优化：滚动只显姓名，定格显示姓名+头衔，缩小后仅姓名
- 支持添加任意数量班级，只需放入新的 CSV 文件
- 旧的 JS 名单文件已移除

### 启动与日志
- 启动日志逐行滚动，12 项详细检查（主题/配置/插件/DOM）
- 每条日志带成功/失败文字说明
- 顶栏新增 📋「系统日志」按钮，可随时查看状态
- 有错误时按钮右上角显示红色错误计数

### 界面优化
- 更新日志弹窗改进：关闭按钮固定在底部，不随内容滚动
- 顶栏版本号移到文字右上方，不可点击
- 📜 更新日志按钮加图标
- 项目根目录整理：文档移入 `docs/`，logo 移入 `main/`

### 工程化
- `server.py` 为文本文件添加 UTF-8 charset 头，解决 CSV 乱码
- 新增 `start.bat` 一键启动脚本
- 新增 `RULES.md` 第 9 条「文件写入规范」（禁用 PowerShell 写文件）
- CSV 解析改为纯 JS，去掉 SheetJS 依赖

## v2.0

### 布局双模式
- 新增列表/网格双模式切换（`☰` / `⊞` 按钮）
- 网格模式：卡片方阵排列，适合投屏浏览
- 列表模式：卡片横向排列，答案跟随题目下方，CSS Multi-Column 原生列优先填充
- 切换模式时伴有过渡动画
- 网格和列表各自独立记忆列数和字号

### 配置系统重构
- 所有默认参数从配置文件读取，零硬编码兜底
- 配置文件统一在 `config/` 目录，按学科分文件夹
- 网格模式、列表模式的列数和字号分别可配置
- 配置文件添加详细中文注释和取值说明

### 英语插件增强
- 默认列表模式，2 列显示
- 按钮「🔍 从服务器获取」进入时自动扫描 `data/english/` 目录
- 支持本地 `.xlsx` / `.csv` 文件导入
- 听写列和答案列可通过步进器调整，切换即时生效

### 化学插件
- 默认网格模式，4 列显示
- 价层电子式和轨道式同时展示
- H~Kr 36 元素，范围选择器

### 界面优化
- 答案文字正常换行（`pre-wrap`），去掉虚线分隔
- 卡片序号跟随字号缩放
- 控制栏紧凑化，增大卡片空间
- 随机点名动画约 2 秒
- 默认浅色主题
- favicon 图标

### 项目结构
- 目录重构：`main/`（代码）、`config/`（配置）、`plugins/`（插件）、`data/`（数据）
- `server.py` 提供通用 `/api/<name>-files` 接口，禁止目录浏览
- URL hash 路由：`#/english`、`#/chemistry`
- 更新日志独立为 `update.md`，点击「更新日志」按钮动态加载

### 开屏动画
- 启动时显示品牌动画
- 逐项检查：配置加载、插件注册、页面元素
- 点击任意位置关闭
