# SeaScribe 更新日志

## v3.0.0

### 代码模块化
- CSS 拆分为 9 个文件（theme / base / topbar / pages / controls / cards / splash / changelog / responsive）
- JS 拆分为 9 个文件（core / theme / navigator / controls / cards / picker / changelog / splash / app）
- `index.html` 瘦身为纯 HTML 骨架，仅保留主题初始化 4 行内联脚本
- 统一命名空间 `window.SeaScribe`
- v2.0 原封不动保留

### 界面优化
- Topbar 去作者名，更新日志移至右侧紧挨夜间模式
- Topbar 版本号缩小为小字灰色
- 控制栏按钮回归单行，紧跟字号控件
- 开屏添加「本项目使用 vibecoding 开发」

### 工程化
- 接入 Git 版本管理，托管于 [GitHub](https://github.com/qxdho/SeaScribe)
- 添加 `.gitignore`（排除 data/、zip、reasonix 等敏感文件）
- v1.0 / v2.0 旧版本源码归档至仓库
- 制定 `RULES.md` 编码规范（8 条规则，含 Git 工作流、版本号管理）
- 遵循语义化版本（SemVer），每次发版打 tag

## v2.0

### 布局双模式
- 新增列表/网格双模式切换（`☰` / `⊞` 按钮）
- 网格模式：卡片方阵排列，适合投屏浏览
- 列表模式：卡片横向排列，答案跟随题目下方，CSS Multi-Column 原生列优先填充
- 切换模式时伴有过渡动画
- 网格和列表各自独立记忆列数和字号

### 配置系统重构
- 所有默认参数从配置文件读取，零硬编码兜底
- 配置文件统一在 `config/` 目录，按学科分文件夹
- 网格模式、列表模式的列数和字号分别可配置
- 配置文件添加详细中文注释和取值说明

### 英语插件增强
- 默认列表模式，2 列显示
- 按钮「🔍 从服务器获取」进入时自动扫描 `data/english/` 目录
- 支持本地 `.xlsx` / `.csv` 文件导入
- 听写列和答案列可通过步进器调整，切换即时生效

### 化学插件
- 默认网格模式，4 列显示
- 价层电子式和轨道式同时展示
- H~Kr 36 元素，范围选择器

### 界面优化
- 答案文字正常换行（`pre-wrap`），去掉虚线分隔
- 卡片序号跟随字号缩放
- 控制栏紧凑化，增大卡片空间
- 随机点名动画约 2 秒
- 默认浅色主题
- favicon 图标

### 项目结构
- 目录重构：`main/`（代码）、`config/`（配置）、`plugins/`（插件）、`data/`（数据）
- `server.py` 提供通用 `/api/<name>-files` 接口，禁止目录浏览
- URL hash 路由：`#/english`、`#/chemistry`
- 更新日志独立为 `update.md`，点击「更新日志」按钮动态加载

### 开屏动画
- 启动时显示品牌动画
- 逐项检查：配置加载、插件注册、页面元素
- 点击任意位置关闭
